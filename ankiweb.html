This add-on add four shortcuts to the reviewer: - open a dialogue box where you chose which tag to add to the card, default key "0" - add a predefined tag(s) to the card. The default key used to be "7" but since 2.1.20 "7" is used by Anki jump forward 5 seconds. - add a predefined tag(s) to the card and move it to a predefined deck, default key "8" - move the card to a predefined deck, default key "9" You can set the predefined tags and deck in the add-on settings via Main Window -> Add-ons - "Add Tag and Move 2.1" -> Config If you find a bug report it at my github repo.
